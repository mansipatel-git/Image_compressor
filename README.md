# 📸 Image Compressor

A simple and efficient **Image Compression Tool** built using Python that reduces image file size while maintaining good visual quality.

---

## 🚀 Features
- 📉 Compress images to reduce file size  
- 🖼️ Supports multiple image formats (JPG, PNG, etc.)  
- ⚡ Fast and efficient processing  
- 💾 Saves compressed images automatically  

---

## 🧠 How It Works
1. Load the input image  
2. Apply compression (resize / quality reduction)  
3. Save the compressed image  
4. Compare original vs compressed size  

---

## 🛠️ Tech Stack
- Python  
- OpenCV  
- NumPy  
- PIL (Python Imaging Library)  

---

## 📊 Example Output
- Original Size: 2.5 MB  
- Compressed Size: 800 KB  
- Compression Ratio: ~68% reduction
